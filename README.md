## ğŸ“Œ Project Overview

Meteorite landings provide valuable insight into near-Earth objects and their historical impact on Earth.  
This project analyzes meteorite discovery records spanning from **year 860 to the present**, enabling both global and long-term exploration of meteorite events.

The dashboard allows users to:
- Explore **global geospatial distributions** of meteorite landings
- Analyze **trends over time** in meteorite discoveries
- Compare the **largest recorded meteorites by mass**
